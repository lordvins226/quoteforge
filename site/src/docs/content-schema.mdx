Every card and deck is a JSON document validated by Zod. Files are auto-detected by their `type` field: `"card"` (or missing) → single card, `"deck"` → carousel.

## Card shape

```json
{
  "type": "card",
  "template": "manifesto",
  "theme": "obsidian",
  "size": "instagram-sq",
  "blocks": [
    { "type": "headline", "text": "Ship typographic cards from JSON." },
    { "type": "text", "text": "Version them. Script them. Deploy them." },
    { "type": "bullet-list", "items": ["Fast.", "Deterministic.", "Yours."] }
  ]
}
```

**Fields:**

| Field | Type | Notes |
|-------|------|-------|
| `type` | `"card"` | Optional — cards are the default |
| `template` | `"manifesto" \| "quote" \| "list" \| "minimal"` | See [Templates](/docs/templates) |
| `theme` | string | Name matching `themes/<name>.json` |
| `size` | SizeName | One of 14 presets (see below) |
| `blocks` | Block[] | Ordered list of content blocks |

## Deck shape

```json
{
  "type": "deck",
  "defaults": {
    "template": "quote",
    "theme": "dawn",
    "size": "instagram-sq"
  },
  "slides": [
    { "blocks": [{ "type": "headline", "text": "Slide 1" }] },
    { "blocks": [{ "type": "headline", "text": "Slide 2" }], "theme": "obsidian" }
  ]
}
```

Each slide inherits `defaults`; any field can be overridden per-slide.

## Block types

Seven block types, each with its own shape:

### `headline`
```json
{ "type": "headline", "text": "Bold statement", "size": "xl" }
```

### `text`
```json
{ "type": "text", "text": "Regular paragraph copy." }
```

### `blockquote`
```json
{ "type": "blockquote", "text": "A quote.", "attribution": "— Author" }
```

### `bullet-list`
```json
{ "type": "bullet-list", "items": ["First point", "Second point"] }
```

### `callout`
```json
{ "type": "callout", "variant": "tip", "text": "Helpful note." }
```

Variants: `tip`, `warning`, `info`, `success`.

### `divider`
```json
{ "type": "divider" }
```

### `spacer`
```json
{ "type": "spacer", "size": "md" }
```

## Sizes

Fourteen presets, all CSS-pixel dimensions:

| Name | Dimensions | Platform |
|------|-----------|----------|
| `twitter` | 1200×675 | Twitter/X post |
| `instagram-sq` | 1080×1080 | Instagram feed |
| `story` | 1080×1920 | IG/FB Story |
| `facebook-post` | 1200×630 | Facebook post |
| `facebook-cover` | 1640×624 | Facebook cover |
| `facebook-square` | 1080×1080 | Facebook carousel |
| `facebook-event` | 1920×1080 | Event cover |
| `facebook-group-cover` | 1640×856 | Group cover |
| `linkedin-post` | 1200×627 | LinkedIn feed |
| `linkedin-article` | 1200×644 | Article cover |
| `pinterest` | 1000×1500 | Pinterest pin |
| `youtube-thumb` | 1280×720 | YouTube thumbnail |
| `og-image` | 1200×630 | Open Graph |
| `custom` | any | User-defined (requires `width`/`height` at root) |

## Validation

Use `bun quoteforge validate <file>` to check any card or deck without rendering. Errors point at the exact path and expected type.
